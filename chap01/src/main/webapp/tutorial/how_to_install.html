2. 아피치-톰캣 설치

  • Tomcat 9.0 설치
  • 압축 해제
  • cmd > cd 해당폴더 > bin 폴더 밑의 startup.bat 실행
  • 설정 파일은 conf폴더 밑의 server.xml (포트번호 수정)
  • webapps폴더 내부에 html파일을 두면 서비스 할 수 있는 프로그램 (서버프로그램)
  • 실행시 JAVA_HOME, JRE_HOME 이라는 이름의 환경변수가 필요하다

3. 프로젝트 생성하기

  • Severs탭 > 서버등록(Tomcat 설치 경로 등록)
  • File - New - Dynamic Web Project
  • 프로젝트 이름 설정
  • 서버선택
  • src/main/webapp에 파일을 두면 서버의 webapps밑에서 동작하게 된다
